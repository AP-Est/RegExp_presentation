Regular Expressions

Author: Andrey Perovskiy

Regular expressions are very useful functionality that can be used in many aspects of programming.

Example

  • Let's take e-mail address
  • UserName@mailService.com
  • and try create validation regular expression for it

    • Let's start

    • separate e-mail on blocks
    • first: "UserName"
    • second: "@"
    • third: "mailService"
    • fourth: "."
    • fifth: "com"

    First block

    • Use "^" for show start of line
    • "\w" any letters "+" one time or more
    • but in address can include more than one word
    • add "." dot and "-" dash
    • use "()" for set conditions and "*" for set count zero or more
    • also use "[]" for set possible symbols and "?" for set count 0 or 1 time
    • unite it "([.-]?\w+)*" - (to first word we add dot or dash 0 or 1 time, add another word) * 0+ times
    • Result: /^\w+([.-]?\w+)*/

    Second block

    • add strict "@" without any adds
    • Result: /^\w+([.-]?\w+)*@/

    Third block

    • Repeat first block
    • Result: /^\w+([.-]?\w+)*@\w+([.-]?\w+)*/

    Fourth block

    • add strict dot ".", since the dot is a system symbol, it cannot simply be specified, there are several ways to escape "\." or specify the second option in the selection most clearly "[.]"
    • Result: /^\w+([.-]?\w+)*@\w+([.-]?\w+)*[.]/

    Fifth block

    • add last 2 or 3 letter of domain
    • use "\w" and "{}" for set count its letters
    • in final use "$" for show end of line
    • Result: /^\w+([.-]?\w+)*@\w+([.-]?\w+)*[.](\w{2,3})$/

    Now we can use this regular expression for validate Email address

    • const regExp = /^\w+([.-]?\w+)*@\w+([.-]?\w+)*[.](\w{2,3})$/
    • const email = 'example@examp.com'
    • const result = regExp.test(email)
    • console.log(result)
